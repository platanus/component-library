# Tailwind CSS Custom Forms

Repository - Documentation

Official-ish library that provides custom CSS for form inputs.

yarn add @tailwindcss/custom-forms -D

// tailwind.config.js
module.exports = {
  // ...
  plugins: [
    require('@tailwindcss/custom-forms')
  ]
